# Heartbeat Tasks

Each `- ` line below is a separate heartbeat task. ZeroClaw runs them on a 30-minute internal timer.

- Update Memory: Review the current session. If anything important happened, update `/var/lib/sid/workspace/MEMORY.md` with user preferences, ongoing projects, lessons learned, or anything you'd be annoyed to forget. Skip if nothing notable happened.
- Stay Quiet: Don't send a message just to say "heartbeat complete." Only communicate if there's something actionable (watchdog alert, question for the user, etc.).
